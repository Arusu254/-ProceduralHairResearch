# 程序化毛发生成器 - 开发手册

**版本**：1.0  
**适用平台**：Unity 2022.3 LTS 及以上  
**开发语言**：C#  
**项目类型**：Unity Editor 扩展工具  

---

## 1. 简介

程序化毛发生成器是一款集成在 Unity 编辑器中的毛发建模工具。用户只需导入 OBJ 格式的头模，通过滑块实时调节发丝数量、长度、卷曲度、粗细以及发根生成高度，即可快速生成自然的头发模型。系统支持**局部区域编辑**（选区删除/恢复发根、独立调节局部参数），并可将最终毛发导出为 OBJ 文件，适用于游戏、影视、虚拟角色等场景。

## 3. 安装与导入

### 3.1 导入项目

将以下文件夹和文件复制到您的 Unity 项目的 `Assets` 目录中：
- `Assets/`
  - `Scripts/`
    - `Core/`
      - `BezierCurve.cs`
      - `HairStrandMesh.cs`
      - `OBJLoader.cs`
      - `OBJExporter.cs`
      - `HairGeneratorCore.cs`
    - `Editor/`
      - `HairGeneratorWindow.cs`
  - `Materials/`
    - `HairMaterial.mat` （会自动生成）
  - `StreamingAssets/` （可选，存放头模 OBJ 文件）

  
### 3.2 首次运行

- 打开 Unity，等待编译完成。
- 点击菜单栏 `Tools → 程序化毛发生成器` 打开主窗口。
- 若 `Assets/Materials/HairMaterial.mat` 不存在，系统会自动创建一个默认棕色材质并启用 GPU Instancing。

---

## 4. 快速开始

### 4.1 导入头模

1. 单击 **“导入 OBJ 头模”** 按钮。
2. 在文件对话框中选择一个 `.obj` 格式的人头模型（建议顶点数 1 万以内）。
3. 导入成功后，场景中会出现 `PreviewHead` 物体（灰色材质），窗口会显示顶点数。

> **注意**：系统会自动将 Z-up 模型旋转为 Y-up，以适配 Unity 坐标系。

### 4.2 生成毛发

1. 调整 **“发丝数量”**（建议从 1000 开始测试性能）。
2. 滑动 **长度**、**卷曲度**、**粗细** 滑块。
3. 根据需要调整 **“发根 Y 值阈值”**（值越大头发越集中在头顶）。
4. 单击 **“生成毛发”** 按钮，稍等片刻即可在 Scene 视图中看到生成的头发。

### 4.3 导出毛发

- 生成毛发后，单击 **“导出为 OBJ”**。
- 选择保存路径和文件名，系统会将当前毛发 Mesh 导出为标准 `.obj` 文件。

## 5. 参数说明

| 参数         | 范围      | 说明                                                         |
| ------------ | --------- | ------------------------------------------------------------ |
| 发丝数量     | 100–10000 | 生成的头发总根数，数值越大性能负担越重。                     |
| 长度         | 0.2–2.5   | 头发的绝对长度（世界单位）。                                 |
| 卷曲度       | 0–2.0     | 控制发丝的弯曲程度，0 = 直发，2.0 = 大波浪 / 螺旋。          |
| 粗细         | 0.005–0.06 | 单根发丝的圆柱半径。                                         |
| 发根 Y 值阈值 | 0.1–0.9   | 控制头发从头模的哪个高度开始生成，值越大头发越集中在头顶。 |

## 6. 局部区域编辑

**功能**：在不影响其他区域的前提下，对指定范围内的头发进行删除、恢复或独立参数调节。

### 6.1 进入选区模式

- 勾选 **“进入选区模式”**。
- 在 Scene 视图中用鼠标左键点击头模表面，会生成绿色半透明球体表示选区范围（可多次点击增加选区）。
- 选区半径可通过 **“选区半径”** 滑块调节。

### 6.2 删除与恢复发根

- **删除选中的发根**：将选区内的所有发丝移除（但数据保留，可恢复）。
- **恢复选中的发根**：将之前删除的选区内发丝重新生成。

### 6.3 局部参数调节

- 先调节 **“局部长度乘数”**、**“局部卷曲乘数”**、**“局部粗细乘数”**。
- 点击 **“应用局部参数到选中的发根”**，选区内的发丝形状会按乘数变化（最终长度 = 全局长度 × 局部长度乘数，其他类似）。
- 单击 **“清除选区”** 可以清除所有高亮选点。

> **提示**：局部参数乘数对发根是否删除/恢复独立存储，相互不影响。

## 7. 性能优化建议

- 生成 5000 根以上发丝时，建议先降低曲线离散度（修改 `HairGeneratorCore.cs` 中 `segments = 6`，`radialSegments = 4`）。
- 如果导出卡顿，可在导出前临时将发丝数量降至 2000，导出后再恢复。
- 使用 GPU Instancing 材质（默认已开启）可大幅提升渲染帧率。
- 避免在编辑器中同时运行其他高负载插件。

## 8. 常见问题 (FAQ)

**Q1：导入 OBJ 后模型是平躺的？**  
A：系统已自动旋转 Z-up 模型为 Y-up。如果仍不正确，请检查模型的原始轴向。

**Q2：毛发穿插进头皮内部怎么办？**  
A：系统已在采样时沿法线向外偏移 0.02 单位。若穿插仍然明显，可在 `HairGeneratorCore.cs` 中增大偏移量（`vertices[i] + n * 0.03f`）。

**Q3：为什么我调整“发根 Y 值阈值”后没有变化？**  
A：该参数只在点击“生成毛发”时生效。如果已经生成过，需再次点击“生成毛发”（强制重新采样）才能看到效果。

**Q4：导出 OBJ 非常慢甚至卡死？**  
A：导出超过 50 万顶点时会弹窗警告，可点击“取消”后降低发丝数量再导出。也可以使用本手册第 7 节中的优化方法。

**Q5：如何重置所有参数到默认值？**  
A：关闭窗口重新打开，或手动将滑块归零（数量 2000，长度 0.8，卷曲 0.3，粗细 0.012，阈值 0.7）。

## 9. 脚本结构简介（开发参考）

| 脚本文件 | 职责 |
| -------- | ---- |
| `BezierCurve.cs` | 三次贝塞尔曲线数学计算。 |
| `HairStrandMesh.cs` | 将曲线离散点转换为圆柱体 Mesh。 |
| `OBJLoader.cs` | 解析 OBJ 文件为 Unity Mesh。 |
| `OBJExporter.cs` | 将 Mesh 导出为 OBJ 文件（带进度条）。 |
| `HairGeneratorCore.cs` | 核心算法：采样、曲线生成、合并 Mesh。 |
| `HairGeneratorWindow.cs` | 编辑器窗口，整合所有 UI 和交互（包括局部选区编辑）。 |

开发者可根据需要修改 `HairGeneratorCore.cs` 中的 `randomStrength`（方向随机强度）、`segments`（曲线平滑度）等参数。
